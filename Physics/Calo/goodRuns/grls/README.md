# Good Run Lists (GRLs)

The current/active GRL files are located directly in this base directory.

Each subdirectory (e.g. `run2auau/`, `run2pp/`, `run3auau/`, etc.) contains older productions and archived versions corresponding to that run and dataset.

## Structure

- Base directory:
  - Holds the latest GRL files for each dataset
- Subdirectories:
  - Contain previous productions and historical GRLs for that specific run

## Available Datasets

- run2auau
- run2pp
- run3auau
- run3oo
- run3pp

Each dataset has:
- A current GRL in the base directory
- Older versions stored in its corresponding subdirectory
